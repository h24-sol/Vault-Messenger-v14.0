# VAULT MESSENGER v14

## 🚀 Automatic Build
This repository is equipped with **GitHub Actions**. 
1. Push any change to the `main` branch.
2. Go to the **Actions** tab in this repo.
3. Click the latest run.
4. Download the **vault-v14-apk** from the "Artifacts" section at the bottom.

## 🛠 Tech Stack
- **Encryption:** Shadow Matrix (Steganography)
- **Database:** Firebase Realtime DB
- **Framework:** CapacitorJS
